## Описание

Проект представляет собой тестирование виджета чата и приложения с этим виджетом. Реализованы:
- Тесты для виджета (рендеринг, открытие/закрытие, выполнение шагов)
- Тесты для приложения (рендеринг, работа формы, взаимодействие с виджетом)

## Как собрать

1. Клонировать репозиторий
2. Установить зависимости: `npm install`
3. Запустить тесты: `npm test`

## Технологии

- JavaScript
- Jest
- Testing Library
- Vite
